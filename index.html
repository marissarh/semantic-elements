Semantic Elements

What are Semantic Elements?

Semantic elements are tags that clearly describe its meaning to both the browser and developer. Instead of using basic tags such as div and span (which tell us nothing about the content that is being added), use semantic HTML5 tags that are descriptive. Examples of such tags are header, footer, nav, and section. Semantic HTML5 elements are used to enhance accessibility, searchability, internationalization, and interoperability.
